a <- 1
b <- 3

typeof(a)

Functions

We can apply a sum function to our variables in different ways:

sum(a,b)
## [1] 4
a + b
## [1] 4
sum(5,6)
## [1] 11

Results from functions can be stored in new variables:

c <- sum(a,b)
(c <- sum(a,b))
## [1] 4

We can then check the value for the new variable or use it for more functions

str(c)
##  num 4
sum(a,c)
## [1] 5

Importantly, the sum function cannot be applied to characters.

a <- "abc"
b <- "hello"
sum(a,b)
## Error in sum(a, b): invalid 'type' (character) of argument
typeof(a)
## [1] "character"
typeof(b)
## [1] "character"
str(a)
##  chr "abc"
str(b)
##  chr "hello"

We can transform them into booleans:

a <- TRUE
b <- FALSE
sum(a,b)
## [1] 1
sum(a,a)
## [1] 2
typeof(a)
## [1] "logical"
typeof(b)
## [1] "logical"

Vectors

We create a vector by creating a collection of individual values of only one type:

vec <- c(1,2,3,4,1,2,3)

For instance, we can say that the values 1, 2, 3, and 4 correspond to the first 4 letters of the alphabet by assigning names:

names(vec) <- LETTERS[1:4]
str(vec)
##  Named num [1:7] 1 2 3 4 1 2 3
##  - attr(*, "names")= chr [1:7] "A" "B" "C" "D" ...

Even if we assigned characters to our values, R can still read them as numbers:

sum(vec)
## [1] 16
mean(vec)
## [1] 2.285714
vec[2]
## B 
## 2
vec[3]
## C 
## 3
vec["A"]
## A 
## 1
vec[c("A", "D")]
## A D 
## 1 4

The unique function returns the non-duplicate values in our vector

unique(vec)
## [1] 1 2 3 4
